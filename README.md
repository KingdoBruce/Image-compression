# 图片压缩工具

一个纯前端的在线图片压缩工具，所有处理均在浏览器本地完成，保护您的隐私安全。

## 功能特性

- 🖼️ **支持多种格式**：JPG、PNG、WebP
- 🔒 **隐私安全**：所有处理在本地完成，图片不会上传到服务器
- ⚡ **批量处理**：支持同时上传和压缩多张图片
- 🎛️ **灵活设置**：可调节压缩质量、最大宽度、输出格式
- 📱 **响应式设计**：支持桌面端和移动端
- 🎨 **现代化界面**：暗色主题，美观易用

## 使用方法

1. 拖拽图片到上传区域，或点击上传区域选择图片
2. 调整压缩设置（质量、最大宽度、输出格式）
3. 点击"开始压缩"按钮
4. 下载压缩后的图片

## 部署到 GitHub Pages

1. Fork 或克隆此仓库
2. 进入仓库设置 → Pages
3. Source 选择 `main` 分支，文件夹选择 `/ (root)`
4. 保存后等待部署完成
